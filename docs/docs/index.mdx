Solito is the missing piece for using React Native with Next.js to build powerful, cross-platform apps.

## About

Today, this library is two things:

1. A tiny wrapper around React Navigation and Next.js' router that lets you share navigation code
2. A set of recommendations and examples for building cross-platform apps with Next.js on Web
